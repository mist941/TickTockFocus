How to Use TickTockFocus

Creating Timer Presets

  1. Go to the Presets tab
  2. Click Create preset
  3. Enter a name for your preset
  4. Add clock segments by setting hours, minutes, seconds
  5. Click Add after each segment
  6. Rearrange segments by dragging if needed
  7. Click Create to save your preset

Running a Timer

  1. Go to the Timer tab
  2. Select a preset from the dropdown
  3. Click Start to begin the timer
  4. The timer will run through all segments
  5. You'll receive notifications at each milestone

Tips

  • Timers continue running even if you close the popup
  • Click on the extension icon to check progress
  • Create different presets for different activities
  • Use the clock in the header to see the current time
